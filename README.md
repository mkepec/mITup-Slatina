## Step-by-step guide (Windows)
1. Install Git for Windows

Go to https://git-scm.com/download/win
Download the installer (it will auto-detect 64-bit or 32-bit)
Run the installer
Use the default settings (just keep clicking "Next")
Verify installation by opening Command Prompt and typing:

```
git --version
```

### 2. Install Python

1. Go to https://www.python.org/downloads/
2. Download Python 3.11 or 3.12 (latest stable version)
3. **Important:** Run the installer and CHECK the box "Add Python to PATH" at the bottom
4. Click "Install Now"
5. Verify installation by opening Command Prompt and typing:
```
python --version
```

### 3. Install Jupyter

Open Command Prompt and run:
```
pip install jupyter
```

### 4. Clone and Run the Tutorial

1. Open Command Prompt
2. Navigate to where you want the tutorial folder:
```
cd Desktop
```
3. Clone the repository:
```
git clone https://github.com/anthropics/prompt-eng-interactive-tutorial.git
```
4. Navigate into the tutorial folder:
```
cd "prompt-eng-interactive-tutorial/Anthropic 1P"
```
5. Start Jupyter:
```
jupyter notebook
